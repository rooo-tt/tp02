qué es una URL?

una url es Uniform Resource Locator es una cadena de texto que especifica dónde puede encontrarse un recurso también es conocido como dirección WEB o enlace. que esta compuesto por protocolo, host, sitio, dominio, path del directorio, documentos y ruta absoluta

qué es una url absoluta?

es una direccion web completa que incluye a todos los componentes protocolo, dominio, ruta y recurso

qué es una url relativa?

es una direccion web que no tiene ni protocolo ni dominio

cuál es la diferencia entre una url relativa y una url absoluta?

la diferencia entre una url relativa y una url absoluta es que basicamente un cuenta con todos los protocolos y dominios, la url relativa no

qué es un directorio? ¿qué relación hay con una carpeta?

el directorio en la url es la ruta de acceso a algun recurso y tiene relacion con las carpetas porque cuando el servidor recibe una solicitud de recursos usa la url para saber donde esta el recurso que le pidio. cada directorio es una carpeta abierta.

el directorio raiz es una carpeta donde se almacenan los archivos del sitio web de un nombre de dominio
se representa con el simbolo "/"

¿Cuál es el propósito de la etiqueta anchor?

el proposito de la etiqueta anchor es hacer enlaces a cualquier url

¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo de la etiqueta anchor es href de un enlace Hypertext REFerence

¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

si el atributo "href" esta vacio no tenes en donde poner la url, por lo tanto no te llevaria a ningun lugar

Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

el uso de la etiqueta target es especificar en donde desplegar la URL enlazada. Es un nombre name of, o palabra clave keyword for, un contexto de navegación browsing context
los posibles valores son:
self: Carga la URL en el mismo contexto de navegación que el actual y el predeterminado.
blank Carga la URL en un nuevo contexto de navegación
parent Carga la URL en el contexto de navegación padre
top Carga la URL en el contexto más alto de navegación
