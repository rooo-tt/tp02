qué es una URL?

una url es Uniform Resource Locator es una cadena de texto que especifica dónde puede encontrarse un recurso también es conocido como dirección WEB o enlace. que esta compuesto por protocolo, host, sitio, dominio, path del directorio, documentos y ruta absoluta

qué es una url absoluta?

es una direccion web completa que incluye a todos los componentes protocolo, dominio, ruta y recurso

qué es una url relativa?

es una direccion web que no tiene ni protocolo ni dominio

cuál es la diferencia entre una url relativa y una url absoluta?

la diferencia entre una url relativa y una url absoluta es que basicamente un cuenta con todos los protocolos y dominios, la url relativa no

qué es un directorio? ¿qué relación hay con una carpeta?

el directorio en la url es la ruta de acceso a algun recurso y tiene relacion con las carpetas porque cuando el servidor recibe una solicitud de recursos usa la url para saber donde esta el recurso que le pidio. cada directorio es una carpeta abierta.
